# 🏠 Grace Real Estate

A full-stack real estate platform enabling users to browse, purchase, and rent properties with secure, integrated payment processing.

🔗 **Live Demo:** [grace-real-estate.vercel.app](https://grace-real-estate.vercel.app)

## Overview

Grace Real Estate was built to simplify property transactions end-to-end — from listing discovery to secure payment — in a single platform rather than scattering the process across multiple disconnected tools.

## Features

- 🏡 Browse property listings with detailed information and images
- 💳 Secure payment processing for purchases and rentals
- 🔐 User authentication and protected transaction flows
- 📊 RESTful API architecture connecting frontend and backend seamlessly
- 📱 Responsive design across devices

## Tech Stack

- **Frontend:** React, TypeScript
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Language:** JavaScript, TypeScript

## Getting Started

### Installation

```bash
git clone https://github.com/hillariousice/gracerealestate.git
cd gracerealestate
```

**Backend:**
```bash
cd server
npm install
npm run dev
```

**Frontend:**
```bash
cd client
npm install
npm run dev
```
